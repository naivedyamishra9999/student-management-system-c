# Student Management System in C

A console-based Student Management System developed in C using Linked Lists and Dynamic Memory Allocation.

## Features

* Add Student Records
* Display All Students
* Search Student by ID
* Search by Specialization
* Update Student Records
* Delete Student Records
* Sort by Marks
* Sort by Name
* Find Highest and Lowest Marks
* Calculate Average Marks
* Count Total Students
* Stack Operations (Pop & Peek)
* Memory Management using Dynamic Allocation

## Technologies Used

* C Programming
* Linked Lists
* Dynamic Memory Allocation
* Structures
* File Handling Concepts
* Data Structures

## Learning Outcomes

* Practical implementation of Linked Lists
* Dynamic memory management using malloc() and free()
* CRUD operations in C
* Sorting and searching algorithms
* Menu-driven application development
